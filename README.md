## 🚀 快速开始

### 1. 安装 Skill

将本仓库克隆到 OpenClaw Skills 目录：

```bash
git clone https://github.com/payudon/unlock-xiaomi-be6500pro-ssh.git ~/.qclaw/skills/unlock-xiaomi-be6500pro-ssh
```

### 2. 使用 Skill

在 OpenClaw 对话中触发：

- "帮我解锁小米 BE6500 Pro 路由器的 SSH"
- "如何在小米路由器上安装 ShellCrash"
- "BE6500 Pro 开启 SSH"

## 📚 操作概述

1. **刷入稳定版固件**（1.0.46）
2. **获取 STOK 密钥**
3. **开启 SSH 端口**
4. **备份固化 SSH**
5. **升级固件**（可选）
6. **安装 ShellCrash**

详细步骤请查看 [SKILL.md](./SKILL.md)。

## ⚠️ 安全提示

**解锁设备有风险，若无编程能力请严格按照教程步骤执行，切勿随意插拔设备，以免设备变砖。**
